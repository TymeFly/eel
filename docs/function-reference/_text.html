Functions that manipulate or generate text.

text.index(find, values...)

Returns the 1-based index of the first Value whose text representation matches the find text. This function can be used with a compound statement to determine the index of a lookback, for example:
$( exp1; exp2; exp3; $[text.index('search', value1, value2, value3)-NotFound] )

Called as:

text.index(find, values...)

Parameters:

find
Type : Text
Description : The value to search for
values...
Type : Value
Description : One or more values to test, in the order they should be evaluated

Return type:

Number

Returns:

The 1-based index of the first matching value, or -1 if no match is found

Since:

3.2

text.random(length, validCharacters)

Generates random text of the specified length from a set of characters, which may be customised. This function does not guarantee that a character is chosen from each range within validCharacters or that all characters in the generated text are unique.

Called as:

text.random()
text.random(length)
text.random(length, validCharacters)

Parameters:

length
Type : Number
Default Value : 10
Description : The length of the returned text; must be greater than 0
validCharacters
Type : Text
Default Value : A-Za-z0-9
Description : The characters that may appear in the returned text. Ranges can be specified using a dash ( -), for example a-z or 0-9. A literal dash ( -) can be included in the set of valid characters so long as it is either the first or last character specified

Return type:

Text

Returns:

Random text of the specified length

Since:

3.0
