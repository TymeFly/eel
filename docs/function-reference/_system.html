system

Functions that are host-specific. For example, this group contains functions that return information about the local file system.

system.fileSeparator()

Returns the operating system path separator as text. This is typically a forward slash ( '/') or a backslash ( '\').

Called as:

system.fileSeparator()

Return type:

Text

Returns:

The operating system path separator as text

Since:

1.0

system.home()

Returns the canonicalised path representing the current user's home directory. The result always includes a trailing system.fileSeparator().

Called as:

system.home()

Return type:

Text

Returns:

The canonicalised path of the current user's home directory, with a trailing file separator

Since:

1.0
