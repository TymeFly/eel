logic

Functions that manipulate or generate logical values.

logic.index(values...)

Returns the 1-based index of the first truevalue in the given sequence. This function can be used with a compound statement to determine the index of a lookback, for example:
$( exp1; exp2; exp3; $[logic.index(value1, value2, value3)-NotFound] )

Called as:

logic.index(values...)

Parameters:

values...
Type : Value
Description : One or more values to test, in the order they should be evaluated

Return type:

Number

Returns:

The 1-based index of the first true value, or -1 if all values are false
