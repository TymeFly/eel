format.at(zone, format, offsets...)

Returns a formatted date in the specified zone with optional offsets, formatted according to the specified format text. This function is equivalent to $format.date(format, date.at(zone), offsets).

Called as:

format.at(zone, format, offsets...)

Parameters:

zone
Type : Text
Description : The time zone.
format
Type : Text
Description : The format of the returned text.
offsets...
Type : Text
Description : Optional offsets to apply.

Return type:

Text

Returns:

The formatted date text.

See:

format.date(format, date, offsets...)
Java DateTimeFormatter

format.binary(value)

Returns the binary representation of the given value as text. The result does not include a leading 0b. This is equivalent to $format.number(value, 2).

Called as:

format.binary(value)

Parameters:

value
Type : Number
Description : The value to convert.

Return type:

Text

Returns:

The number in binary text.

See:

2.1.0

format.number(value, radix)

Returns the representation of the given value as text in the specified radix.

Called as:

format.number(value, radix)

Parameters:

value
Type : Number
Description : The value to convert.
radix
Type : Number
Description : The radix of the text representation. This must be in the range 2 to 36, inclusive.

Return type:

Text

Returns:

The number as text in the specified radix.

Throws:

IllegalArgumentException If the radix is out of range.
