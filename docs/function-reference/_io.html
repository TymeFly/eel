io

Functions that read data from local files.
To guard against potential denial-of-service attacks, a limit is imposed on the amount of data each function can read. By default, this is 32768 bytes, but it can be changed in the EEL context. If a function attempts to read beyond this limit, an IOException is thrown.

io.head(file, lines)

Returns up to lines lines from the beginning of the specified UTF-8 text file. Lines are delimited by a line feed ( \n), a carriage return ( \r), a carriage return followed by a line feed, or by reaching the end of the file.

Called as:

io.head(file)
io.head(file, lines)

Parameters:

file
Type : Text
Description : The file on the local file system to read from
lines
Type : Number
Default Value : 1
Description : The maximum number of lines to return

Return type:

Text

Returns:

Up to lines lines from the beginning of the file

Throws:

IOException If the file cannot be read, or the head exceeds the length defined by the context

Since:

3.0

io.tail(file, lines)

Returns up to lines lines from the end of the specified UTF-8 text file. Lines are delimited by a line feed ( \n), a carriage return ( \r), a carriage return followed by a line feed, or by reaching the end of the file.

io.tail(file)
io.tail(file, lines)

Parameters:

file
Type : Text
Description : The file on the local file system to read from
lines
Type : Number
Default Value : 1
Description : The maximum number of lines to return

Return type:

Text

Returns:

Up to lines lines from the end of the file

Throws:

IOException If the file cannot be read or exceeds the length defined by the context

Since:

3.0
