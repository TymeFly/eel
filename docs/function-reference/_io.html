io

Functions that read data from local files.

To guard against potential denial-of-service attacks, a limit is imposed on the amount of data each function can read. By default, this is 32768 bytes, but it can be changed in the EEL context. If a function attempts to read beyond this limit, an IOException is thrown.

io.head(file, lines)

Returns up to count lines from the start of the given UTF-8 text file. Lines are delimited with \n

Called as:

io.head(file)
io.head(file, lines)

Parameters:

file
Type : Text
Description : The file on the local file system.
lines
Type : Number
Default Value : 1
Description : The maximum number of lines to read.

Return type:

Text

Returns:

Up to count lines from the start of the file.

Throws:

IOException If the file could not be read.

Since:

3.0
