General utilities group
Name Type Description
abs(value) Number Returns the absolute value of the specified number using the precision defined by the context.
accessedAt(file, defaultValue) Date Returns the 'last access time' of the specified file.
acos(value) Number Calculates the arccosine (inverse cosine) of the specified value.
after(text, delimiter, count) Text Returns all text following the count'th occurrence of the specified delimiter.
afterFirst(text, delimiter) Text Returns all text following the first occurrence of the specified delimiter.
afterLast(text, delimiter) Text Returns all text following the last occurrence of the specified delimiter.
asin(value) Number Calculates the arcsine (inverse sine) of the specified value.
atan(value) Number Calculates the arctangent (inverse tangent) of the specified value.
avg(first, others...) Number Returns the average (arithmetic mean) of the specified numbers using the precision defined by the context.
baseName(path, extension) Text Returns the path with any leading directory components removed.
before(text, delimiter, count) Text Returns all text preceding the count'th occurrence of the specified delimiter.
beforeFirst(text, delimiter) Text Returns all text preceding the first occurrence of the specified delimiter.
beforeLast(text, delimiter) Text Returns all text preceding the last occurrence of the specified delimiter.
between(text, delimiter, start, end) Text Returns all text between the start'th and end'th occurrence of the specified delimiter.
char(codepoint) Text Returns a single-character text value corresponding to the specified Unicode codepoint.
codepoint(text) Number Returns the numeric Unicode code point of the first character in the specified text.
contains(text, search) Number Returns the number of times the specified search phrase occurs in the text.
cos(value) Number Calculates the cosine of the specified value.
count(name) Number Returns the next integer value in an incrementing counter.
createAt(file, defaultValue) Date Returns the 'creation time' of the specified file.
dirName(path) Text Returns the path with its last non-slash component and trailing slashes removed.
duration(from, to, period) Number Returns the difference between two dates in a specific period of time.
exists(path) Logic Returns true only if the specified file exists.
exp(value) Number Returns the natural exponent of the specified value using the precision defined by the context.
extension(path, max) Text Returns the file extensions from the filename in the specified path, including any leading dot ('.') characters.
extract(text, regEx) Text Extracts characters from the specified text using a Java-style regular expression.
factorial(value) Number Computes the factorial of the specified value using the precision defined by the context.
fail(message) An EEL function that fails the expression by throwing an EelFailException.
fileCount(directory, glob) Number Returns the number of files in the specified directory that match the glob pattern.
fileSize(file, defaultValue) Number Returns the length of the specified file in bytes.
firstAccessed(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was most recently accessed.
firstCreated(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was created first.
firstModified(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was modified least recently.
indexOf(text, subText, defaultValue) Number Returns the 0-based index of the first occurrence of subText in the specified text, or defaultValue if subText does not occur.
isBlank(text) Logic Returns true if the specified text is empty or contains only whitespace.
isEmpty(text) Logic Returns true if the specified text is empty.
lastAccessed(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was least recently accessed.
lastCreated(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was created most recently.
lastIndexOf(text, subString, defaultValue) Number Returns the 0-based index of the last occurrence of subString in the specified text, or defaultValue if subString does not occur.
lastModified(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was modified most recently.
left(text, length) Text Returns a portion of the specified text based on the given length.
len(text) Number Returns the length of the specified text, including leading and trailing spaces.
ln(value) Number Returns the natural logarithm (base e) of the specified value using the precision defined by the context.
log(value) Number Returns the base-10 logarithm of the specified value using the precision defined by the context.
lower(text) Text Converts all the characters in the supplied text to lower case.
matches(text, regEx) Logic Checks whether the specified text matches the given Java-style regular expression.
max(first, others...) Number Returns the largest value among the specified numbers.
mid(text, position, length) Text Returns the middle portion of the specified text.
min(first, others...) Number Returns the smallest value among the specified numbers.
modifiedAt(file, defaultValue) Date Returns the 'last modified time' of the specified file.
padLeft(text, width, pad) Text Left-pads the specified text with the given character until it reaches the specified width.
padRight(text, width, pad) Text Right-pads the specified text with the given character until it reaches the specified width.
printf(format, arguments...) Text Generates formatted text using the specified format string and arguments, similar to Java's format(String, Object[]).
random(min, max) Number Returns a random number between the specified min and max, inclusive.
realPath(path) Text Returns the path in canonical format based on the current operating system.
replace(text, from, to) Text Replaces all occurrences of the literal from in the specified text with the literal to.
replaceEx(text, regEx, to) Text Replaces each part of the specified text that matches the Java-style regular expression regEx with the literal text to.
right(text, length) Text Returns a portion of the specified text based on the given length.
root(value, base) Number Returns the n-th root of the specified value using the precision defined by the context.
sgn(value) Number Returns the sign of the specified numeric value.
sin(value) Number Calculates the sine of the specified value.
tan(value) Number Calculates the tangent of the specified value.
title(text) Text Converts all the characters in the supplied text to title case.
toDegrees(value) Number Converts the specified radian value to degrees using the precision defined by the context.
toRadians(value) Number Converts the specified degree value to radians using the precision defined by the context.
trim(text) Text Returns the text with leading and trailing spaces removed.
upper(text) Text Converts all the characters in the supplied text to upper case.
uuid() Text Returns a new pseudo-randomly generated UUID using a cryptographically strong pseudo-random number generator.

Eel group
Name Type Description
eel.buildDate() Date Returns the date and time the EEL compiler was built.
eel.version() Text Returns the EEL version as text in the format major.minor, where both parts are numeric.

System group
Name Type Description
system.fileSeparator() Text Returns the operating system path separator as text.
system.home() Text Returns the canonicalised path representing the current user's home directory.
system.pwd() Text Returns the canonicalised path of the current working directory.
system.temp() Text Returns the canonicalised path of the temporary directory.

Text group
Name Type Description
text.index(find, values...) Number Returns the 1-based index of the first Value whose text representation matches the find text.
text.random(length, validCharacters) Text Generates random text of the specified length from a set of characters, which may be customised.

Number group
Name Type Description

number.ceil(number) Number Returns the smallest integer value that is greater than or equal to the given number.
number.e() Number Returns the value of e calculated to the precision defined by the context.
number.floor(number) Number Returns the largest integer value that is less than or equal to the given number.
number.pi() Number Returns the value of pi calculated to the precision defined by the context.
number.round(number, precision) Number Returns the number rounded to the specified precision.
number.truncate(number, precision) Number Returns the number truncated to the specified precision.

Logic group
Name Type Description
logic.index(values...) Number Returns the 1-based index of the first truevalue in the given sequence.

Date group
Name Type Description
date.at(zone, offsets...) Date Returns the current date/time in the specified zone with optional offsets applied.
date.local(offsets...) Date Returns the current date/time in the local time zone with optional offsets applied.
date.minus(date, offsets...) Date Returns a copy of the given date with all specified offsets subtracted.
date.moveZone(date, zone) Date Returns a copy of the given date with a different time zone.
date.parse(pattern, date, offsets...) Date Parses text representing a date/time using the specified DateTimeFormatter pattern.
date.plus(date, offsets...) Date Returns a copy of the given date with all specified offsets added.
date.set(date, spec...) Date Returns a copy of the given date with individual fields updated according to the specified spec.
date.setZone(date, zone) Date Returns a copy of the given date with a different time zone.
date.start(zone, offsets...) Date Returns the date/time at which the given context was created, adjusted to the specified time zone.
date.utc(offsets...) Date Returns the current date/time in the UTC time zone, with optional offsets applied.

log.info(first, others...) Value Writes a message to the com.github.tymefly.eel.log logger at the INFO level.
log.trace(first, others...) Value Writes a message to the com.github.tymefly.eel.log logger at the TRACE level.
log.warn(first, others...) Value Writes a message to the com.github.tymefly.eel.log logger at the WARN level.

Format group
Name Type Description

format.binary(value) Text Returns the binary representation of the given value as text.
format.date(format, date, offsets...) Text Returns the given date formatted according to the specified format and optional offsets.
format.hex(value) Text Returns the hexadecimal representation of the given value as text.

format.start(format, zone, offsets...) Text Returns the date/time at which the given context was created, formatted according to the specified format and adjusted to the specified zone with optional offsets.
format.utc(format, offsets...) Text Returns the current date/time, formatted according to the specified format and adjusted to the UTC zone with optional offsets.
