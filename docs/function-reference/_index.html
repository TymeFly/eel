General utilities group
Name Type Description
abs(value) Number Returns the absolute value of the specified number, with the precision specified by the context.
accessedAt(file, defaultValue) Date Returns the 'last access time' of the specified file.
acos(value) Number Calculates the arccosine (inverse cosine) of the specified value.
after(text, delimiter, count) Text Returns all the text after the count'th occurrence of the delimiter text.
afterFirst(text, delimiter) Text Returns all the text after the first occurrence of the delimiter text.
afterLast(text, delimiter) Text Returns all the text after the last occurrence of the delimiter text.
asin(value) Number Calculates the arcsine (inverse sine) of the specified value.
atan(value) Number Calculates the arctangent (inverse tangent) of the specified value.
avg(first, others...) Number Returns the average of a set of numbers.
baseName(path, extension) Text Returns the path with any leading directory components removed.
before(text, delimiter, count) Text Returns all the text before the count'th occurrence of the delimiter text.
beforeFirst(text, delimiter) Text Returns all the text before the first occurrence of the delimiter.
beforeLast(text, delimiter) Text Returns all the text before the last occurrence of the delimiter text.
between(text, delimiter, start, end) Text Returns all the text between the start'th and the end'th occurrence of the delimiter text.
char(codepoint) Text Returns a single-character text value corresponding to the specified Unicode codepoint.
codepoint(text) Number Returns the numeric Unicode code point of the first character in the specified text.
contains(text, search) Number Returns the number of times that the search phrase occurs in the text.
cos(value) Number Calculates the cosine of the specified value.
count(name) Number Returns the next integer value in an incrementing counter.
createAt(file, defaultValue) Date Returns the 'creation time' of the specified file.
dirName(path) Text Returns the path with its last non-slash component and trailing slashes removed.
duration(from, to, period) Number Returns the difference between two dates in a specific period of time.
exists(path) Logic Returns true only if the specified file exists.
exp(value) Number Returns the natural exponent of value with the precision specified by the context.
extension(path, max) Text Returns the file extensions from the filename in the specified path, including any leading dot ('.') characters.
extract(text, regEx) Text Extracts characters from the specified text based on a Java Regular Expression.
factorial(value) Number Computes the factorial of the specified value with the precision specified by the context.
fail(message) An EEL function that fails the expression by throwing an EelFailException.
fileCount(directory, glob) Number Returns the number of files in the specified directory that match the glob pattern.
fileSize(file, defaultValue) Number Returns the length of the specified file in bytes.
firstAccessed(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was most recently accessed.
firstCreated(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was created first.
firstModified(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was modified least recently.
indexOf(text, subString, defaultValue) Number Returns the 0-based index of the first occurrence of subString in the specified text, or the defaultValue if subString does not occur.
isBlank(text) Logic Returns true if the specified text is empty or contains only whitespace.
isEmpty(text) Logic Returns true only if the specified text is empty.
lastAccessed(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was least recently accessed.
lastCreated(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was created most recently.
lastIndexOf(text, subString, defaultValue) Number Returns the 0-based index of the last occurrence of subString in the specified text, or the defaultValue if subString does not occur.
lastModified(directory, glob, index, defaultValue) Text Returns the full path to the file in the specified directory that was modified most recently.
left(text, length) Text Returns the first portion of the specified text according to the given length.
len(text) Number Returns the length of the specified text, including any leading and trailing spaces.
ln(value) Number Returns the natural logarithm (base e) of the specified value with the precision specified by the context.
log(value) Number Returns the logarithm base 10 of the specified value with the precision specified by the context.
lower(text) Text Converts all the characters in the supplied text to lower case.
matches(text, regEx) Logic Checks whether the specified text matches a Java regular expression.
max(first, others...) Number Returns the largest value in a set of numbers.
mid(text, position, length) Text Returns the middle portion of the specified text.
min(first, others...) Number Returns the smallest value in a set of numbers.
modifiedAt(file, defaultValue) Date Returns the 'last modified time' of the specified file.
padLeft(text, width, pad) Text Returns the specified text with pad characters added to the start so that it is at least width characters long.
padRight(text, width, pad) Text Returns the specified text with pad characters added to the end so that it is at least width characters long.
printf(format, arguments...) Text Generates formatted text using the specified format string and arguments, similar to Java's format(String, Object[]).
random(min, max) Number Returns a random number between min and max, inclusive.
realPath(path) Text Returns the path in canonical format based on the current operating system.
replace(text, from, to) Text Replaces all instances of the literal from in the specified text with the literal to.
replaceEx(text, regEx, to) Text Replaces each part of the specified text that matches the Java regular expression regEx with the literal text to.
right(text, length) Text Returns the last portion of the specified text according to the given length.
root(value, base) Number Calculates the n’th root of the specified value.
sgn(value) Number Returns the sign of a numeric value.
sin(value) Number Calculates the sine of the specified value.
tan(value) Number Calculates the tangens of the specified value.
title(text) Text Converts all the characters in the supplied text to title case.
toDegrees(value) Number Returns the specified radian value expressed in degrees.
toRadians(value) Number Returns the specified degree value expressed in radians.
trim(text) Text Returns the text with any leading and trailing spaces removed.
upper(text) Text Converts all the characters in the supplied text to upper case.
uuid() Text Returns a new pseudo-randomly generated UUID, created using a cryptographically strong pseudo-random number generator.

Eel group
Name Type Description
eel.buildDate() Date Returns the date and time the EEL compiler was built.
eel.version() Text Returns the EEL version as text in the format major.minor, where both parts are numeric.

System group
Name Type Description
system.fileSeparator() Text Returns the operating system path separator as text.
system.home() Text Returns the canonicalised path representing the current user's home directory.
system.pwd() Text Returns the canonicalised path of the current working directory.
system.temp() Text Returns the canonicalised path of the temporary directory.

Text group
Name Type Description
text.random(width, validCharacters) Text Generates random text of the specified length, using a set of valid characters which may be customised.

Number group
Name Type Description
number.c() Number Returns the speed of light in metres per second.
number.ceil(number) Number Returns the smallest integer value that is greater than or equal to the given number.
number.e() Number Returns the value of e to the precision specified by the context.
number.floor(number) Number Returns the largest integer value that is less than or equal to the given number.
number.pi() Number Returns the value of pi to the precision specified by the context.
number.round(number, precision) Number Returns the number rounded to the specified precision.
number.truncate(number, precision) Number Returns the number truncated to the specified precision.

Date group
Name Type Description
date.at(zone, offsets...) Date Returns the current date in the specified zone with optional offsets applied.
date.local(offsets...) Date Returns the current date in the local time zone with optional offsets applied.
date.minus(date, offsets...) Date Returns a copy of the given date with all specified offsets subtracted.
date.moveZone(date, zone) Date Returns a copy of the given date with a different time zone.
date.plus(date, offsets...) Date Returns a copy of the given date with all specified offsets added.
date.set(date, spec...) Date Returns a copy of the given date with individual fields updated according to the specified spec.
date.setZone(date, zone) Date Returns a copy of the given date with a different time zone.
date.start(zone, offsets...) Date Returns the DATE at which the given context was created in the UTC time zone.
date.utc(offsets...) Date Returns the current date in the UTC zone with optional offsets applied.

Log group
Name Type Description
log.debug(first, others...) Value Writes a message to the logger at the DEBUG level.
log.error(first, others...) Value Writes a message to the logger at the ERROR level.
log.info(first, others...) Value Writes a message to the logger at the INFO level.
log.trace(first, others...) Value Writes a message to the logger at the TRACE level.
log.warn(first, others...) Value Writes a message to the logger at the WARN level.

Format group
Name Type Description
format.at(zone, format, offsets...) Text Returns a formatted date in the specified zone with optional offsets, formatted according to the specified format text.
format.binary(value) Text Returns the binary representation of the given value as text.
format.date(format, date, offsets...) Text Returns the given date formatted according to the specified format and optional offsets.
format.hex(value) Text Returns the hexadecimal representation of the given value as text.
format.local(format, offsets...) Text Returns a formatted date in the local time zone with optional offsets, formatted according to the specified format text.
format.number(value, radix) Text Returns the representation of the given value as text in the specified radix.
format.octal(value) Text Returns the octal representation of the given value as text.
format.start(format, zone, offsets...) Text Returns a formatted date with optional offsets, formatted according to the specified format and zone.
format.utc(format, offsets...) Text Returns a formatted date in the UTC zone with optional offsets, formatted according to the specified format text.

Io group
Name Type Description
io.head(file, lines) Text Returns up to count lines from the start of the given UTF-8 text file.
io.tail(file, lines) Text Returns up to count lines from the end of the given UTF-8 text file.
