General utilities

Functions in the general utility group provide common operations for text, number, logic, and date types. As these functions are frequently used, they are given short names without a group prefix.

printf(format, arguments...)

Generates formatted text using the specified format string and arguments, similar to Java's format(String, Object[]). Supported conversion characters include:
  • Text:s, S
  • Character:c, C - the first character of the text argument
  • Logic:b, B
  • Integral Number:d, o, x, X
  • Real Number:e, E, f, g, G
  • Dates:t, T
  • Character literals:%n
All Java formatting indexes, flags, widths, and precisions are supported.

Called as:

printf(format, arguments...)

Parameters:

format
Type : Text
Description : The format string
arguments...
Type : Value
Description : The arguments referenced by the format specifiers in format; extra arguments are ignored if more arguments are provided than format specifiers

Return type:

Text

Returns:

The formatted text

See:

Oracle Java 17 Formatter documentation

Since:

1.1
