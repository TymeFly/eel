Functions that interact directly with the EEL RunTime.

eel.buildDate()

Returns the date and time the EEL compiler was built.

Called as:

eel.buildDate()

Return type:

Date

Returns:

The EEL build date and time

Since:

2.0.0

eel.version()

Returns the EEL version as text in the format major.minor, where both parts are numeric. This format can be easily converted to a number if required.

Called as:

eel.version()

Return type:

Text

Returns:

The version text

Since:

2.0.0
